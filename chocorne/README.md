# Choc Corne

## Case

A case for my slightly misshapen choc corne. The keyboard is based on [this PCB](https://github.com/ceoloide/corney-island), but I generated the files wrong and ordered a choc keyboard with MX spacing so none of the nice cases I wanted to use would fit.

Inspired by the case for the [TOTEM](https://github.com/GEIGEIGEIST/TOTEM), I included cutouts for the hotswap sockets to minimize the height of the board. This case has almost the same typing height as a bare PCB. Because of these cutouts, the case is not symmetrical, so be sure to print each half, and not just mirror the first half. The tolerances are pretty tight so you may have to tweak some of the offsets to make the hotswap sockets and diodes fit in their recesses.

## Carrying Case

I wanted a flat carrying case, maintaining the low profile philosophy. Many cases out there stack the cornes on top of each other, but I preferred keeping them next to each other and making the case the same form-factor as a small tablet so it can fit in my bag by my laptop.

It's just a little too large for my A1 Mini, so I had to do some joinery. Reach out to me if you want a single-piece version.
